# DrawJS

Vector graphics for modern browsers `>=ES6`

Install:

```shell
yarn add @thesabbir/drawjs
```

or 

```shell
npm i @thesabbir/drawjs
```

CDN:
```html
<script type="module" src="https://unpkg.com/@thesabbir/drawjs@latest/release/drawjs.js"></script>
```
